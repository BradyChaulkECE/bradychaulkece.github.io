Outlined

A tool that converts course outline PDFs into structured Obsidian study vaults.

Store Page GitHub Repository

Overview

Outlined is a tool that transforms course outline PDFs into fully structured Obsidian course vaults. University course outlines contain essential information such as units, assignments, and important dates, but students typically have to manually organize this information for studying. Outlined automatically extracts these sections from a syllabus and generates a ready-to-use study vault with organized notes, unit folders, and structured templates for efficient exam preparation.

The Problem

Students often rely on course outlines as the central reference for their semester, yet the information is typically locked inside a static PDF. Extracting useful study structure from a syllabus requires manually creating folders, organizing units, listing assignments, and copying key dates. This process is repetitive and time-consuming, especially when managing multiple courses.

The Solution

Outlined automates this workflow. The program analyzes the course outline PDF and extracts key academic structures such as units, assessments, and dates. It then generates a structured Obsidian vault with predefined note templates designed for organized studying. Students can immediately begin working inside a structured knowledge system without manual setup.

Output Structure

For each course outline, Outlined generates a structured vault containing:

  • Course Overview
  • Assessment Breakdown
  • Important Dates
  • Unit folders with structured study templates

Each unit contains a "Unit Summary" template with sections for:

  • Core Concepts
  • Definitions
  • Equations
  • Problem-Solving Strategies
  • Exam Questions

How It Works

The workflow is designed to be simple and fast:

  • Launch the Outlined application
  • Select a course outline PDF
  • Choose an output folder
  • The tool automatically generates the course vault

The resulting folder structure can be opened directly in Obsidian for immediate use as a study workspace.

Limitations

Course outlines vary significantly in formatting across institutions. Outlined performs best when the document contains identifiable sections such as instructor information, units, assignments, and dates. In some cases minor manual edits may be required after vault generation.
